## *Java Variable* 

#### *What is a variable in cs programming?*
* varible = reusable container for value.
* behaves = same as the value it contains.

### *Java Varable Data Type*
---
#### Primitive vs Reference

* Primitive = simple value stored directly in memory (stack)

* Reference = memory address (stack) that points to (heap) that address stack.
---
#### *Primitive Data Type vs Reference Data Type* 
---
|Sr. no.| Primitive | Reference |
| :--: | :----------: | :----------: |
| 1  | int    | String|
| 2  | double | array |
| 3  | char   | object|   
| 4  | boolean|      |   

> [!CAUTION]
> *Primitive* has more *data type* and it's *size* info which *should not be skipped*.
>>i.e. byte, short, long, float .etc

### *Steps to create Variable*
---

1. Declartion
2. Assignment
---
1. Declartion
```java
1. int age;
2. System.out.print(age);
//-> variable not initialized (aka assigned)
```

2. Assignment (asmgt = decl + asgmt)
```java
1. int age = 21;
2. System.out.print(age);
//-> age variable will be printed here
```
