# Java Introduction

## What is Java?
Java is a programming language used to build applications which can be used
across devices and os. (-Due to JRE/JVM)

It is:
- Object-oriented
- Platform independent (runs anywhere via JVM)
- Widely used in backend systems, Android apps, and enterprise software

---

## Why learn Java?
- Strong OOP foundation
- Industry usage is still very high
- Helps in understanding backend systems

---

## Core Idea of Programming
Programming is about solving problems using structured instructions.

In Java, we solve problems using:
- Classes
- Objects
- Methods
- Variables

---

## Key takeaway
Java programs are built using objects that interact with each other.
